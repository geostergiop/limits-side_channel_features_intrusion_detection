The downloadable code package excludes data/traffic.db because the full artifact with the bundled SQLite database was triggering UI download failures.

Recommended path:
1. Unzip the code package.
2. Place your pcaps under mnt/benign/ and mnt/attack_pcap/ as before.
3. Set OPENAI_API_KEY in your shell.
4. Rebuild the database so the new flow timeout / TCP boundary logic is applied:
   python run_all.py --phase 1 --rebuild-db
5. Run the rest of the phases as needed.

If you want to keep using an existing traffic.db instead of rebuilding it from pcaps, the fixed code can still use it. However, the improved sessionization logic only affects newly extracted data.
